Requisitos
Estudios mínimos: Licenciado - Técnico
Experiencia mínima: 1 año
Imprescindible residente en: Barcelona
Conocimientos informáticos: 
Se requiere programador con buen nivel de programación C# con Visual Studio para el desarrollo de aplicaciones desktop, webs y webservices (mínimo 2 años de experiencia). Es necesario dominar HTML5, javascript con jQuery, el framework de desarrollo ASP.NET, y la gestión de base de datos mediante Microsoft SQL Server.
Se valorarán conocimientos de Microsoft MVC, del gestor de contenidos Umbraco y de Adobe Flash.
Idiomas : Castellano / catalán; muy valorado conocimientos de inglés e italiano
Requisitos deseados: 
Conocimiento a nivel técnico de plataformas de ecommerce : Prestashop y Magento
Conocimiento de CMS : Umbraco, WordPress o Drupal
