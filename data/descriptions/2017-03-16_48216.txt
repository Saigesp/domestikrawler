
Estamos buscando Frontend Developers que nos ayuden a construir la UI de una plataforma cloud de tecnología punta, desde el principio y utilizando lo último en tecnologías de frontend. En esta posición, en la que no faltarán los desafíos, esperamos que junto a un equipo de personas con talento lideres la innovación en la nube desarrollando una UI de alta calidad y aplicando metodologías de desarrollo Agile.
Las tareas y responsabilidades de este puesto incluyen: 
-	Desarrollar, documentar y desplegar código de alta calidad para implementar la UI web de la plataforma. 
-	Participar en reuniones prescritas por los procesos Agile aplicados, colaborando en las tareas de extracción de requisitos, análisis y diseño entre otras. 
-	Mantenerse actualizado en las últimas tecnologías de frontend. 
-	Escribir test automatizados que validen la plataforma ante los casos de uso y proporcionen un buen coverage del código frontend de la misma. 
-	Coordinar y evaluar procesos para identificar, reportar, realizar seguimiento, escalar y resolver problemas de la plataforma desarrollada. 
-	Servir de consultor técnico para otras unidades de negocio. 
-	Ser proactivo en la resolución de problemas.
Habilidades técnicas: 
-	Mínimo 4 años de experiencia en el desarrollo frontend.
-	Mínimo 2 años de experiencia en el desarrollo web de UI para escritorio/móvil.
-	HTML5, CSS3.
-	SASS.
-	Javascript: AngularJS, Flux/Redux - React o plataformas/librerías similares.
-	Experiencia desarrollando aplicaciones SPA construidas en base a APIs REST.
-	Metodologías Agile.
Habilidades adicionales: 
-	Nodejs.
-	Sistemas de building como grunt, gulp, webpack, etc.
-	Conocimientos de programación funcional y orientada a objetos.
-	ECMAScript v6, JQuery, GWT.
-	Fundation/Bootstrap, LESS, PostCSS.
-	Experiencia con Git o cualquier otro sistema de control de versiones.
-	Experiencia integrando OAuth.
-	Experiencia aplicando metodologías TDD/BDD con herramientas similares a Cucumber and Ginkgo/RSpec/Jasmine.
-	UX.
-	Conceptos de seguridad: SQL Injection, XSS, CSRF, etc.
-	Experiencia en el desarrollo Backend.
-	Experiencia con virtualización, contenedores o servicios cloud IaaS.
-	Contribuciones a proyectos open-source y/o repositorios públicos en Github con tu trabajo.
-	Titulación universitaria en Ingeniería Informática u otra ingeniería.
-	Nivel alto de inglés, tanto hablado como escrito.
-	Alta motivación.
-	Disponibilidad para viajar.
-	Preparado para trabajar en equipo.
