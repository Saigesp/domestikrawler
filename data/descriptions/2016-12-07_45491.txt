
Descripción de empleo
Como desarrollador/a de front-end / JavaScript tendrás la responsabilidad de centrar la programación para crear y mejorar la experiencia de usuario. El cometido principal será la creación de interfaces para todo tipo de dsipositivos, para ofrecer la mejor usabilidad. Hay un gran reto en crear componentes muy reactivos.
Buscamos un alto nivel de implicación y entusiasmo por el desarrollo de aplicaciones robustas orientadas a la usabilidad. Acostumbrado a la creación de prototipos y pruebas piloto con usuarios para la posterior integración con las validaciones necesarias.
También estarás trabajando mano a mano con nuestro equipo de desarrollo back-end, usando sus interfaces REST para acceder a las funcionalidades y datos fundamentales de nuestra plataforma. Creemos en un equipo dinámico, donde cada integrante comparta sus ideas y conocimientos, y todos aprendan de sus habilidades y experiencia. Nos organizamos con la metodología SCRUM.
RESPONSABILIDADES • Creación de nuestras aplicaciones front-end • Asegurando una mayor eficiencia para dispositivos móviles y sobremesa • Escribir código robusto para facilitar los trabajos/experiencia del usuario final • Proveer código JavaScript, HTML5, CSS3 testeado, idiomático y documentado • Coordinando con nuestros desarrolladores back-end para la creación de interfaces REST • Comunicación e integración con servicios externos web (API, WebSocket) • Escribir y mantener unit-testing básico (Travis y Browserstack) • Automatizando procesos de compresión y simplificación de los paquetes finales HABILIDADES Y CUALIFICACIONES • Habilidad para JavaScript y HTML5/CSS3 • Conocimiento de creación de aplicaciones basadas en un framework Javascript (React, Polymer, AngularJS, etc.) • Experiencia en el uso de Canvas y WebGL • Asegurando una cadena clara de dependencias (en lógica programática y estructura de ficheros) • Familiarizado con el uso de interfaces REST y WebSocket • Escribir código que no bloquee la carga en los navegadores; Optimización de uso de la memoria y carga red • Familiaridad montando entornos web de desarrollo (Docker) • Documentación en linea usando JSDoc u otras convenciones • Experiencia laboral en desarrollo demostrable con proyectos reales • Preferible un título relacionado con el sector • Español (hablar/escribir); Inglés (lo suficiente para lectura de documentación)
