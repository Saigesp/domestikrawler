- Amplios conocimientos XHTML, CSS
   ¿Cuándo fue la última vez que utilizaste un tabla para maquetar? ;)

   ¿Sabes cómo optimizar html para que google lea mejor?
- Amplios conocimientos Javascript + frameworks (MooTools, jQuery, Prototype)
   ¿Bendijiste el día en el que se acabó hacer una versión del script para cada navegador?
   ¿Sabes cuando dom ready y cuando load? 
   ¿Serías capaz de calcular el tiempo que te has ahorrado utilizándolos?
- Amplios conocimientos de PHP
   ¿Hacen falta palabras?
 MySQL
