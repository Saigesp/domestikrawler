- HTML + CSS + js (jQuery) (mínimo 5 años de experiencia). Además se valorará el conocimiento angular.js y otros frameworks de desarrollo de javascript. 
- PHP (mínimo 3 años de experiencia) , se valorará el conocimiento de frameworks de desarrollo.
- MYSQL ( mínimo 3 años de experiencia) 
- Mobile web y Responsive Design (2 años de experiencia) 
- Se valorará el desarrollo de webs con CMS de Drupal o Wordpress (mínimo 2 años de experiencia) 
- Se valorará conocimiento de servidores, repositorios de versiones Git.
- Se valorará conocimientos de UI/UX
Buscamos a una persona para incorporarse a la plantilla a jornada completa de manera presencial y de manera indefinida.
