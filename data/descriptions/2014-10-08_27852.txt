-	Dominio del HTML5 + CSS3 para aplicaciones Web 2.0 y 3.0.
-	Amplia experiencia JavaScript y jQuery.
-	PHP nivel alto para el desarrollo de aplicaciones de gestión de contenidos.
-	Experiencia en el desarrollo de interfaces de usuario basadas en tecnología web.
-	Experiencia en desarrollo de aplicaciones Orientadas a Objetos.
Se valorará:
-	Dominio del lenguaje ActionScript para aplicaciones FLASH y FLEX.
-	Experiencia en desarrollo de aplicaciones para móviles.
-	Experiencia en desarrollo de WebApps para plataformas iOS.
-	Experiencia en desarrollo en lenguajes tradicionales (C#, Java...)
No se responderán candidaturas de empresas o estudios, buscamos un trabajador a tiempo completo para formar parte de nuestro equipo de desarrollo base.
