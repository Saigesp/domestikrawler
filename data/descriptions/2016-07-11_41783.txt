A continuación comentamos algunos de sus aspectos tecnológicos más interesantes:
- Cloud Azure architecture
- Multi region replication
- Razor engine markup

Ser curioso, entusiasta y autónomo.

-	Gran dominio de ASP.NET y entity framework

-	Experiencia previa en Azure cloud
-	Experto en SQL server administración
