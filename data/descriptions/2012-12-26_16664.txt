
Buscamos una persona que le apasione el desarrollo web/móvil que busque cambiar el mundo a través de la tecnología, y que le interese formar parte de un gran equipo emprendedor de un producto con una gran tracción inicial ya probada.
Se trabajará empleando metodologías ágiles de desarrollo de software. La exigencia del puesto es alta, pues el impacto y la experiencia que se obtendrá será muy grande. Se valorará aquellos candidatos abiertos a aprender y trabajar con nuevas plataformas.
Para nuestra plataforma web buscamos programadores con experiencia contrastada en las siguiente tecnologías:
* Javascript
* AJAX
* HTML
* CSS
* PHP
* Conocimientos de Bases de Datos con MySQL.
Para nuestra plataforma móvil buscamos programadores con experiencia contrastada en las siguientes tecnologías móviles:
* iOS (SDK iOS, XCode)
* Android (SDK Android, Java)
* Desarrollo móvil con HTML 5
