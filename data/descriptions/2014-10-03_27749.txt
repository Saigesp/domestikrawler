
Buscamos Maquetador/programador web  con un nivel alto de Wordpress, que controle de PHP, JavaScript (jQuery), HTML5 y CSS3, diseño responsive y con cierta sensibilidad en diseño.
El/la candidata/a ideal debe poseer conocimientos en: 
- CSS3 
- HTML5 
- IQuery 
- WordPress 
- PHP 
- Responsive Web Design (RWD) 
- MySQL 
En cuanto a experiencia, valoraremos: 
- Creación de themes para WordPress 
- HTML: Código limpio y semántico 
- CSS3 
- Responsive Web Design
- Diseño web: Buscamos una persona que sepa hacer buenos acabados, actualizado y con conocimientos de diseño. 
- Programación en PHP/mySQL: al menos debes saber hacer lo fundamental para trabajar. Mucho mejor si eres experto/a.
Es necesario residencia en Granada dado que el puesto es presencial y experiencia demostrable en desarrollo.
